# HunterMussel Website

Static site built with Docusaurus for SEO-first delivery (pre-rendered HTML).

## Stack

- Docusaurus
- React
- Markdown blog content
- Cloudflare Pages

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
```

Build output directory: `build/`

## Deploy (Cloudflare Pages)

```sh
npm run deploy
```

Wrangler config uses:

- `pages_build_output_dir = "build"` in `wrangler.toml`

## Content

- Homepage: `src/pages/index.tsx`
- Blog posts: `blog/*.md`
- Static assets: `static/`

## SEO

- Static HTML generated by Docusaurus on build
- Sitemap generated by Docusaurus preset
- Global Organization JSON-LD defined in `docusaurus.config.mjs`
