This article is also a self-criticism.

We are shipping faster than ever.
Every week, I learn about a new model, tool, or supposedly a new default. or a new way to code and deliver apps, I mean, to do the exact same thing we've been doing for about 15 years. each new tool with a promise to be the new default.
I've heard about: how to get rich with GPT, or how I did a full autonomous IA agent developer, that works 24/7 for free forever, or even "this is the model that will fully replace humans", but there are two things I'm sure of. I never had coded so fast before, and never got worse code than now.

Okay, but if those tools are so good and can improve my time, avoiding the need to spend hours reading documentation, forums, and testing ways to do things, then why is my code getting worse?
